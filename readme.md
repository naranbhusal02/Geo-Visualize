## This application serves several map and geo related tools to be used for general purpose and helps to visualize our location

This web-based app integrated multiple geo-tools in a single location making it easier to user for accessing multiple serviecs through a single platform.

**_The homepage has a container with links to the other pages of the tools to be used by users as you can see in the preview below:_**

# Tools available

├── Get your coordinates
├── Visualize coordinates
├── Distance Calculator
Preview

**File Structure for the Geo-Visualize Project**

```bash
├── distance/        #Geo tool to get distance between 2 selected points
├── locate-you/      #Geo tool to visualize you longitude and latitude in a map
├── track-location/  #Geo tool to get coordinates of your location
├── index.html       #Main page with links to all the pages and tools
├── script.js
├── style.css        #CSS Style code for main home page
```
